<h1>Antidote</h1>

Antidote is a customized Hugo theme based on [Roxo](https://github.com/StaticMania/roxo-hugo).

## Features

* Responsive Ready.
* Powered by Bootstrap 4.
* Dedicated portfolio page.
* Blog Support.
* Well formatted code.
* Easy Customization.
* Formspree Contact form.
* Crafted for Design Agency/ Portfolio
